# @naxmefy/blueprint

## Installation
```
$ npm install --global @naxmefy/blueprint
```

## Usage
```
// this loads blueprint.yml per default
$ blueprint user/repo

// - or use custom blueprint -

$ blueprint user/repo/my-custom-blueprint.yml
```
